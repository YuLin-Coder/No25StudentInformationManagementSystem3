## 项目介绍

基于servlet的学生信息管理系统：前端 html、jquery、layui，后端 servlet，一个简洁的学生信息、教师信息管理系统

## 主要功能

- 基本功能：登录、退出、注册
- 学生信息管理：根据学号查询、学生列表、添加学生、学生信息修改、学生信息删除、学生信息导出
- 教师信息管理：根据姓名查询、教师列表查询、添加教师、教师信息修改、教师信息删除
